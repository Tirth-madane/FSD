# Experiment 5 – Bootstrap Images & News Article Portal

A responsive webpage demonstrating Bootstrap 5 grid layout for side-by-side image cards and a fully structured HTML news article page with a dynamic reading progress bar.

## 🛠 Tech Stack

- HTML5
- CSS3
- Bootstrap 5 (loaded via CDN — **no installation needed**)
- Vanilla JavaScript

## 📋 Prerequisites

No installations required. Only a modern web browser with an internet connection is needed (Bootstrap is loaded from CDN).

## 🚀 How to Run

1. Open the project folder:
   ```
   Fsd_Codes/5_bootstrap_images_and_news/
   ```

2. Double-click `index.html` to open it directly in your browser.

   **OR** right-click `index.html` → *Open with* → Choose your browser (Chrome, Firefox, Edge, etc.)

> ⚠️ **Internet Required:** Bootstrap 5 is loaded from a CDN. Make sure you have an active internet connection for full styling.

## 📁 Project Structure

```
5_bootstrap_images_and_news/
├── index.html    # Main HTML (image grid + news article)
└── style.css     # Custom styles (on top of Bootstrap)
```

## ✅ Features

- 3 responsive image cards in a Bootstrap column grid
- Hover zoom animation on image cards
- Structured news article with author, date, and blockquote
- Scroll-based reading progress bar at the top of the page
