# Experiment 6 – Feedback Form & Navigation Menu

A static webpage demonstrating a responsive horizontal navigation bar and a styled feedback form with toast notification on submission.

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## 📋 Prerequisites

No installations required. Only a modern web browser is needed.

## 🚀 How to Run

1. Open the project folder:
   ```
   Fsd_Codes/6_feedback_form_and_nav/
   ```

2. Double-click `index.html` to open it directly in your browser.

   **OR** right-click `index.html` → *Open with* → Choose your browser (Chrome, Firefox, Edge, etc.)

## 📁 Project Structure

```
6_feedback_form_and_nav/
├── index.html    # Main HTML (navbar + feedback form)
└── style.css     # Stylesheet
```

## ✅ Features

- Responsive horizontal navigation bar with styled links
- Centered feedback form card with Name, Email, and Comments fields
- Form validation (HTML5 `required` attributes)
- Success toast notification displayed on form submission (no page reload)
