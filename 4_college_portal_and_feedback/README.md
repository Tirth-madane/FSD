# Experiment 4 – College Portal & Feedback System

A static college information portal demonstrating HTML tables, lists, forms, and AJAX-based feedback submission — all without a backend server.

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (AJAX simulation)

## 📋 Prerequisites

No installations required. Only a modern web browser is needed.

## 🚀 How to Run

1. Open the project folder:
   ```
   Fsd_Codes/4_college_portal_and_feedback/
   ```

2. Double-click `index.html` to open it directly in your browser.

   **OR** right-click `index.html` → *Open with* → Choose your browser (Chrome, Firefox, Edge, etc.)

## 📁 Project Structure

```
4_college_portal_and_feedback/
├── index.html    # Main HTML (portal + feedback form)
├── style.css     # Stylesheet
└── app.js        # JavaScript for AJAX feedback handling
```

## ✅ Features

- Academic programs and tuition fee table
- Admission guidelines list
- Campus gallery section
- Admissions enquiry form with validation
- AJAX-based student feedback hub (dynamic, no page reload)
