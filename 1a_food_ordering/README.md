# Experiment 1a – Online Food Ordering System

A static front-end food ordering app where users can add menu items to a cart, view totals, and place an order.

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (inline)

## 📋 Prerequisites

No installations required. Only a modern web browser is needed.

## 🚀 How to Run

1. Open the project folder:
   ```
   Fsd_Codes/1a_food_ordering/
   ```

2. Double-click `index.html` to open it directly in your browser.

   **OR** right-click `index.html` → *Open with* → Choose your browser (Chrome, Firefox, Edge, etc.)

## 📁 Project Structure

```
1a_food_ordering/
├── index.html    # Main HTML page (menu, cart, order logic)
└── style.css     # Stylesheet
```

## ✅ Features

- Browse a food menu with items and prices
- Add items to cart dynamically
- View cart with subtotal, delivery charge, and total
- Place order with confirmation alert
