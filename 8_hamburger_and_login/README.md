# Experiment 8 – Responsive Navbar & Animated Login Form

A static webpage demonstrating two UI components: a responsive hamburger navigation menu (for mobile screens) and a login form with client-side validation and shake animation on error.

## 🛠 Tech Stack

- HTML5
- CSS3 (animations, responsive media queries)
- Vanilla JavaScript

## 📋 Prerequisites

No installations required. Only a modern web browser is needed.

## 🚀 How to Run

1. Open the project folder:
   ```
   Fsd_Codes/8_hamburger_and_login/
   ```

2. Double-click `index.html` to open it directly in your browser.

   **OR** right-click `index.html` → *Open with* → Choose your browser (Chrome, Firefox, Edge, etc.)

## 📁 Project Structure

```
8_hamburger_and_login/
├── index.html    # Main HTML (navbar + login form)
└── style.css     # Stylesheet (hamburger animation, form styles)
```

## ✅ Features

- **Hamburger Menu:** Responsive nav bar that collapses into a hamburger icon on small screens; click to toggle the menu open/closed
- **Login Form Validation:**
  - Email format check
  - Password minimum length (6 characters)
  - Shake animation triggered on invalid submission
  - Green border highlight on valid credentials
